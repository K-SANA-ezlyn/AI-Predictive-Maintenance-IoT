# AI-Powered Predictive Maintenance (NASA CMAPSS)

## Overview
This project predicts machine failures using jet engine sensor data.

## Dataset
NASA CMAPSS Dataset

## Features
- 21 sensor readings
- Time-series engine data

## Approach
- Convert RUL → classification
- Train Random Forest model

## Output
- Failure prediction
- Accuracy score
- CSV output

## Run
python main.py
